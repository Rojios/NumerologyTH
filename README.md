# NumerologyTH (เลขศาสตร์ TH)

iOS app วิเคราะห์เลขศาสตร์ประยุกต์ไทย

## Features
- วิเคราะห์เบอร์โทรศัพท์ (Mode A) — คะแนน 1,000 คะแนน
- วิเคราะห์ชื่อ-นามสกุลภาษาไทย (Mode B)
- วิเคราะห์ทะเบียนรถยนต์ (Mode D)

## Requirements
- iOS 17.0+
- Xcode 15.0+

## Architecture
- SwiftUI + SwiftData
- Local-only (no backend)
- StoreKit 2 for IAP (฿99 one-time unlock)
